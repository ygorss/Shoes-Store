<h1 align="center"><br>
    📦Treino de Flexbox
</h1>


<h4 align="center">Projeto desenvolvido com o objetivo de treinar como utilizar o "display: flex" no CSS.</h4>


---


## 📄 As páginas

O site não tem responsividade, porém possui duas páginas que utilizamos o flexbox:

- **Index**
- **Trabalhe Conosco**
